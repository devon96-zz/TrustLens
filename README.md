# Room distribution
- Alarm - Blue cable - Pin 11
- Bathroom - Orange cable - Pin 10
- Fridge - Green cable - Pin 9
- Lounge - Purple cable - Pin 8
- Kid's room - Yellow cable - Pin 7
- Button - Pin 12
